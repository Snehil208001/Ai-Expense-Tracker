# AI Expense Tracker - Design System

Clean fintech + AI modern feel. Card-based, minimal, premium SaaS look.

## Colors

| Token | Hex | Usage |
|-------|-----|-------|
| Primary | `#4F46E5` | Trust, fintech, CTAs |
| Accent | `#10B981` | Income, positive money |
| Danger | `#EF4444` | Expenses, alerts |
| Background | `#F8FAFC` | App background |
| Card | `#FFFFFF` | Cards, surfaces |

## Typography

- **Balance/numbers**: Large (36px), bold
- **Headings**: Medium (18–24px), semiBold
- **Body**: Base (16px), regular
- **Metadata**: Small (12–14px), muted

## Components

- **Card**: Rounded (16px), soft shadow, padded
- **Button**: Primary (indigo), Secondary (emerald), Outline, Ghost
- **AITipCard**: Left border accent, icon badge, actionable tone
- **ExpenseItem**: Category icon circle, title, date, amount (green/red)

## Screens

1. **Dashboard**: Greeting, balance card, income/expense cards, budget bar, recent expenses, AI tip card, FAB
2. **Add Expense**: Large amount input, category chips, date selector, notes, save CTA
3. **Expense List**: Category icon, title, date, amount (familiar UX)
4. **Scan Receipt**: Camera → Processing → Result (AI branding, editable fields)
5. **Analytics**: Placeholder (bar, pie, trend, filters)
6. **Profile**: Placeholder
